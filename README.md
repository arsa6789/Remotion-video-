# Remotion Video Project

**Created by ShadowARSA**

## 🚀 Quick Start

### 1. Install Dependencies
```bash
npm install
```

### 2. Start Development Server
```bash
npm run dev
```

### 3. Render Video
```bash
npx remotion render
```

## 📺 Preview
Open [http://localhost:3000](http://localhost:3000) after running `npm run dev`
